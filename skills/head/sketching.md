# Sketching

Draw your ideas — starting with quick rough sketches that capture a thought, growing into technical drawings someone else could build from.

## Depth levels

**1 — Introductory.** Draw a solution you would like to build. It only needs to be clear enough that you can explain your idea while pointing at the drawing.

**2 — Early.** Draw a solution you would like to build, and list the materials it needs. Label the parts so a teammate could understand it without you talking.

**3 — Preparatory.** Create a technical drawing of your planned solution, with material labels and short descriptions of what each part does.

**4 — Intermediate.** Create a technical drawing of your planned solution, including measurements, materials, and descriptions of function — accurate enough to build from.

**5 — Advanced.** Create a set of technical drawings showing your solution from multiple views, with measurements, materials, and functions — clear enough for someone else to build without asking you questions.
