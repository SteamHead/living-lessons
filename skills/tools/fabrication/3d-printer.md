# 3D Printer

Start off downloading premade design files and using the machine, later design your own!

## Depth levels

**1 — Introductory.** *Not introduced at this level.*

**2 — Early.** *(AI draft — review)* Watch a 3D print in progress with a mentor. Spot the layers forming, and explain what the printer is doing and why it takes so long.

**3 — Preparatory.** Download a premade 3D model and print it with a mentor's assistance. Retrieve the pieces, and re-tape the printer bed for the next student.

**4 — Intermediate.** Load a 3D model, run the print, retrieve a success, and prepare the printer for the next student. Troubleshoot common issues like adhesion and stringing.

**5 — Advanced.** Maintain and troubleshoot a 3D printer for several weeks. Write training instructions so other students in your school can learn to use it.
