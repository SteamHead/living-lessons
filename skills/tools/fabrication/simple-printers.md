# Simple Printers

Give students control! Large scale, tiled, new media. Responsible and effective use of this tool is where to start.

## Depth levels

**1 — Introductory.** *Not introduced at this level.*

**2 — Early.** *(AI draft — review)* With teacher help, print a single page for your project and collect it from the printer. Learn where prints come out and how to claim only yours.

**3 — Preparatory.** Create a poster in a simple program like Word or PowerPoint. Print with a teacher's help, cut the margins, and assemble a large-format poster.

**4 — Intermediate.** Use a printer independently for several projects. Print on different paper sizes and color settings, and research what ink and paper actually cost.

**5 — Advanced.** Handle printing end to end: install printer drivers, configure settings, run a multi-sheet print, cut the margins, and assemble a large poster.
