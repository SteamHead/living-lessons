# Block Circuits

Build a guided project, open project, and solo invention using littleBits or similar.

## Depth levels

**1 — Introductory.** *(AI draft — review)* Build with snap-together bricks or blocks: first a model of a real-world object, then something you design yourself.

**2 — Early.** Using littleBits or a similar kit, complete a simple guided project. Then experiment freely, with more guided projects available whenever you want them.

**3 — Preparatory.** Using littleBits or similar, plus paper and adhesives to enhance the form, complete a guided project and then a design challenge.

**4 — Intermediate.** Using Grove Zero or similar with paper and adhesives, complete a guided project, explore the project library in free play, then take on a design challenge.

**5 — Advanced.** Using Lego Mindstorms, MakeBlock, or similar, complete a longer guided project — commonly about three hours — then apply the kit to a design challenge.

## Notes

Formerly named **Construction Kits** — that older ladder (Lego bricks, linear blocks, MakeBlock, custom kits) is retired in favor of this one.
