# CNC Drill

Utilize computer-controlled drilling machines (at least 3 axes) to create precise and complex holes and shapes in materials like wood and metal.

## Depth levels

**1 — Introductory.** *Not introduced at this level.*

**2 — Early.** *Not introduced at this level.*

**3 — Preparatory.** *(AI draft — review)* Watch a mentor run the CNC drill on a prepared file. Identify the axes and the safety zones, and describe what the machine is doing.

**4 — Intermediate.** *(AI draft — review)* With a mentor, load a prepared cutting file, secure the material, and run the job. Retrieve the piece and clean the machine for the next student.

**5 — Advanced.** *(AI draft — review)* Prepare a simple file for the CNC drill, run the job with supervision, and troubleshoot a common issue (bit change, re-zeroing). Write tips for the next user.
