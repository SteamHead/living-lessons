# Microprocessors

micro:bit, Arduino, Raspberry Pi, Jetson Nano, Edison.

## Depth levels

**1 — Introductory.** *Not introduced at this level.*

**2 — Early.** *(AI draft — review)* Plug in a classroom microprocessor (micro:bit or similar) and run a program someone else made. Point out the board's main parts.

**3 — Preparatory.** *(AI draft — review)* Load your own simple block-code program onto a micro:bit or similar, to light LEDs or play sounds.

**4 — Intermediate.** *(AI draft — review)* Use a microprocessor with at least one input and one output in a guided project, then modify it in a design challenge.

**5 — Advanced.** *(AI draft — review)* Choose the right board for a project (micro:bit, Arduino, Raspberry Pi), wire it, program it, and install it somewhere it actually gets used.
