# Robotics Coding

Use sensors and actuators to create cars, robots, machines.

## Depth levels

**1 — Introductory.** *Not introduced at this level.*

**2 — Early.** Physical computing: act out the sequence of a program in physical-computing lessons.

**3 — Preparatory.** Use micro:bit or similar to activate LEDs and MIDI sounds, and accept button or motion input. Complete a guided project, and apply it to a real environment (play a game with others, mount it on a door, sew it to a glove).

**4 — Intermediate.** Use block programming, with motors and LEDs as outputs and external sensors as inputs (magnetic, light, touch). Complete a multi-hour guided project to learn basic skills, then a design challenge.

**5 — Advanced.** Use S4A, Arduino, Python for micro:bit, or similar, with motors and LEDs as outputs and external sensors as inputs. Complete a multi-hour guided project, then a design challenge.
