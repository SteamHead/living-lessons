# Electronic Sensors

Boards and the sensors they read — micro:bit and similar: run code on a board, sense the world, and make outputs respond.

## Depth levels

**1 — Introductory.** *Not introduced at this level.*

**2 — Early.** *(AI draft — review)* Plug in a classroom board (micro:bit or similar) and run a program someone else made. Point out the board's main parts, try its built-in sensors — buttons, shake, sound, light — and say what each one detects.

**3 — Preparatory.** *(AI draft — review)* Load your own simple block-code program onto a micro:bit or similar so one sensor input triggers one output — a shake shows a picture, a button plays a sound. Complete a guided tutorial from start to finish.

**4 — Intermediate.** *(AI draft — review)* Use a board with at least one sensor input and one output in a guided project, then modify it in a design challenge — adjusting the sensitivity or the logic until it works reliably.

**5 — Advanced.** *(AI draft — review)* Choose the right board and sensors for your own design (micro:bit, Arduino, Raspberry Pi), wire and program them, calibrate them in the real environment where the project will live, and record what you changed and why.
