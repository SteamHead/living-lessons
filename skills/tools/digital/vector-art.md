# Vector Art

Use 2D vector editing programs to create designs including art, infographics, and design elements.

## Depth levels

**1 — Introductory.** *Not introduced at this level.*

**2 — Early.** Using touchscreens and a simple drawing program, complete a guided project and then a design challenge. Practice undo, layers, and saving your work.

**3 — Preparatory.** Using a laptop and mouse with a simple drawing program, complete a guided project and then a design challenge.

**4 — Intermediate.** Using a vector drawing program, complete a guided project, a download-and-hack project, and then a design challenge.

**5 — Advanced.** Using a vector drawing program, replicate a real-world object to scale, then design a custom object to scale — ready for printing or laser cutting.
