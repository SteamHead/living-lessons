# Hexagon Media

Curriculum images for hexagons — diagrams, tool photos, reference stills. These
are kid-free and public (D3/D6). One folder per hexagon: `media/<hex-id>/`.

- **Images:** drop a file here and reference it from the hexagon markdown, e.g.
  `![LED legs macro](media/hex-simple-circuits/led-legs-macro.jpg)`. Obsidian
  and GitHub both render it inline; the Living Lessons app shows it as a slide.
- **Videos:** never stored here (too heavy for git). Paste a YouTube/Vimeo link
  in the markdown; the app renders it as a video slide.
- **Class photos of students:** do NOT go here. They live in the private R2
  store with face-blur (D4) and appear in Class Journal screens, not curriculum.

The two files in hex-simple-circuits/ are dashed-border PLACEHOLDERS — replace
with real photos, keeping the filenames, and the slides fill automatically.
