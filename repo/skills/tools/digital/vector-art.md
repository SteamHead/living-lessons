# Vector Art

Use 2D vector editing programs to create designs including art, infographics, and design elements.

## Depth levels

**1 — Introductory.** *Not introduced at this level.*

**2 — Early.** Equipment: touchscreens and a simple drawing program. Projects: guided, then design challenge.

**3 — Preparatory.** Equipment: laptop + mouse, and a simple drawing program. Projects: guided, then design challenge.

**4 — Intermediate.** Equipment: laptop + mouse, and a vector drawing program. Projects: guided, download-and-hack, then design challenge.

**5 — Advanced.** Equipment: laptop + mouse, and a vector drawing program. Projects: replicate a real-world object to scale, then create a custom object to scale.
