# Card Craft

Manipulating cardboard, foam core, and card stock is a core prototyping skill! Start simple, and keep adding projects (forever).

## Depth levels

**1 — Introductory.** *Not introduced at this level.*

**2 — Early.** Equipment: scissors, tape, regular paper, cutting templates. Projects: guided, then design challenge.

**3 — Preparatory.** Equipment: prior + recycled media (cups, plates, bags). Projects: guided, then design challenge.

**4 — Intermediate.** Equipment: prior + cardboard, hot glue, cardboard knives. Projects: guided, design challenge, custom design.

**5 — Advanced.** Equipment: prior + laser cutter. Projects: guided, design challenge, custom design, custom kit.
