# Power Stitch Craft

Electric sewing machines are fun! Make tool belts, pillows, etc.

## Depth levels

**1 — Introductory.** *Not introduced at this level.*

**2 — Early.** *(AI draft — review)* With an adult guiding the machine, feed fabric through a sewing machine along a marked straight line.

**3 — Preparatory.** *(AI draft — review)* Thread the machine with help, and sew straight and curved practice lines. Make a single-seam project like a small pillow.

**4 — Intermediate.** *(AI draft — review)* Thread and operate the machine independently. Make a made-to-fit project from a template (bag with straps, simple hat), modifying the template.

**5 — Advanced.** *(AI draft — review)* Design and sew a custom multi-piece project, choosing stitches and fabrics. Do basic machine care (bobbin, needle change) and teach a classmate the basics.
