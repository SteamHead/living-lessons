# Power Carpentry

Create simple projects using electric tools! Drills, sanders, saws? Be careful!

## Depth levels

**1 — Introductory.** *Not introduced at this level.*

**2 — Early.** *(AI draft — review)* With one-on-one adult guidance, use a kid-safe power tool (electric screwdriver or lightweight drill) on a pre-marked practice piece. Say the safety rules out loud first.

**3 — Preparatory.** *(AI draft — review)* Choose the right bit or sandpaper for the job, and complete a guided project using an electric drill or sander with an adult nearby.

**4 — Intermediate.** *(AI draft — review)* Use a supervised stationary tool (scroll saw or drill press) in a guided project that is part of a larger design challenge.

**5 — Advanced.** *(AI draft — review)* Plan and build a project that uses two or more power tools, and write or record the safety steps for each tool so another student could learn from you.
